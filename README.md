# alcatraz_android

A new Flutter project.

## Primeros Pasos

### Requisitos Previos

- [Flutter SDK](https://docs.flutter.dev/get-started/install)
- [VS Code](https://code.visualstudio.com/) con la [extensión de Flutter](https://marketplace.visualstudio.com/items?itemName=Dart-Code.flutter) instalada.

### Ejecución en VS Code

1. Abre la carpeta del proyecto en VS Code.
2. Asegúrate de tener un dispositivo seleccionado en la barra de estado (esquina inferior derecha). Puedes hacer clic en el nombre del dispositivo para seleccionar un emulador, simulador o dispositivo físico conectado.
3. Presiona `F5` para iniciar la depuración, o ve a la vista **Ejecutar y Depurar** (`Ctrl+Shift+D`) y haz clic en **Iniciar depuración**.
4. La aplicación se compilará y se ejecutará en el dispositivo seleccionado.

## Probado en

- **Google Pixel 9** (Android)
